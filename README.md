🔒 Masking Data NIK Menggunakan Excel
Deskripsi
Proyek ini berfokus pada penyamaran (masking) data NIK menggunakan Microsoft Excel untuk menjaga privasi informasi. Sebagian data ditampilkan, sementara sisanya disembunyikan.

Dataset
Data terdiri dari:
- Nama
- NIK
- NIK yang telah disamarkan

Tools & Fungsi yang Digunakan
- Microsoft Excel
- Fungsi:
  - LEFT (mengambil sebagian karakter dari kiri)
  - Operator "&" (menggabungkan teks)

Hasil Analisis
- NIK berhasil disamarkan dengan menampilkan sebagian angka
- Data tetap dapat dikenali tanpa membuka seluruh informasi
- Keamanan data lebih terjaga

Tujuan Proyek
- Memahami konsep masking data
- Melatih penggunaan fungsi teks di Excel
- Meningkatkan kesadaran akan keamanan data

Insight
Masking data merupakan teknik sederhana namun penting untuk melindungi informasi sensitif agar tidak disalahgunakan.

Penutup
Proyek ini merupakan latihan dasar dalam pengolahan data dengan memperhatikan aspek keamanan dan privasi.
